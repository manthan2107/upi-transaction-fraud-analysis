# UPI Fraud Transaction Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on UPI transaction data to uncover fraud patterns, analyze transaction behavior, and generate business-driven insights. Using Python-based analytics and interactive visualizations, the project identifies high-risk transaction characteristics that can support fraud monitoring and serve as a foundation for predictive fraud detection systems.

---

## Problem Statement

With the rapid growth of digital payments, identifying fraudulent UPI transactions has become increasingly important. The objective of this project is to analyze historical transaction data, understand the factors associated with fraud, and derive actionable insights that can improve risk assessment and decision-making.
